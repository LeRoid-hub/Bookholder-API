# Bookholder-API

## Setup
For this project a PostgreSQL database is required.

### Enviroment Vars
```
DB_USER = user
DB_PASSWORRD = password
DB_HOST = my.host
DB_PORT = 5432
DB_NAME = databasename # Optinal; bookholder is the default
SECRET = your32charactersecret
PORT = 8080 # Optional; 8080 is the default port
```
